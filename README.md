# Системный анализ и машинное моделирование

Лабораторные работы для BSUIR/БГУИР (Белорусский государственный университет информатики и радиоэлектроники).
Предмет - SAaMMod (Systems analysis and machine modeling) / САиММод (Системный анализ и машинное моделирование).

## **Лабораторная работа №1**

– Создать генератор последовательности равномерно распределенных случайных чисел на основе смешанного алгоритма Лемера. Подобрать параметры генератора, который дает максимальный период.
– Для сгенерированной выборки построить гистограмму (20+ интервалов).
– Используя критерии хи-квадрат и КС, подтвердить гипотезу о равномерности последовательности.
– Отобразить результаты графически.
– Сделать вывод о согласованности аналитических расчетов и визуализации.

## **Лабораторная работа №2**

1. Для сложной системы, определяемой вариантом задания, построить концептуальную модель и программную имитационную модель.
2. Провести верификацию имитационной модели путем постановки имитационного эксперимента с повторными прогонами.
3. Отобразить результаты имитации с помощью диаграмм.
Требования по реализации
1. Согласовать концептуальную модель с преподавателем.
2. Представить динамику взаимодействия элементов модели в виде квазипараллельного процесса.
3. Запрещается использовать алгоритм с постоянным временным шагом.
4. Выбрать для сложной системы 4+ отклика, из них 1+ в дискретном времени, 1+ в непрерывном времени.
5. Разрешается для всех случайных потоков событий в программе ИМ использовать единственный стандартный для выбранного языка программирования датчик псевдослучайных чисел.
6. Предусмотреть режим трассировки состояния объекта моделирования.
7. Исходные данные для имитационной модели задавать в файле или в коде.

## **Лабораторная работа №3**

Для стационарного состояния имитационной модели сложной системы согласно ЛР2 решить следующие задачи по двум откликам. 
1. Используя критерий хи-квадрат, проверить гипотезу о нормальности распределения откликов при «длинных» повторных прогонах.
2. Вычислить точечные и интервальные оценки откликов ИМ в опыте из 20 повторных прогонов при уровне значимости 0.05.
3. Оценить зависимость точности имитации от количества прогонов и определить, при каком количестве прогонов достигается 5% погрешность отклика.
4. Построить зависимость изменения отклика в модельном времени, выдвинуть и проверить гипотезу об уменьшении времени прогона, исключая переходный период.
5. Проверить гипотезу о возможности постановки опыта с непрерывным прогоном.
6. Оценить чувствительность откликов к вариациям переменных ИМ, используя опыт с непрерывным прогоном (определить достаточную точность задания переменной).
Отобразить результаты решения задач с помощью диаграмм.

## **Лабораторная работа №4**

Для имитационной модели сложной системы согласно ЛР2 решить следующие задачи.
1. Построить зависимость отклика от варьирования параметра модели на 7+ уровнях, выполнить линейную и нелинейную (любую) аппроксимацию, сделать вывод о наилучшем приближении.
2. Определить максимальное количество отказанных ресурсов, при котором объект моделирования возвращается в стационарное состояние.
3. Реализовать эксперимент по сравнению трёх альтернатив использования объекта моделирования.
4. Поставить двухфакторный эксперимент (4+ уровня для каждого фактора), отобразить поверхность отклика, интерпретировать результат (какой параметр является более значимым).
Отобразить результаты решения задач с помощью диаграмм.
